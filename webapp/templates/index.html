信息安全密码算法接口化实现与可测验证平台

基于 Python + Flask 构建，从零实现密码算法，提供 RESTful API 接口与可视化执行轨迹追踪。

🔑 对称加密 豊 哈希与派生 📄 编码转换 🔐 公钥与签名
🔑

对称加密

支持 AES-128 (ECB/CBC, PKCS7)、SM4、RC6 算法，可设置密钥、IV、工作模式与填充方式，Level 2 Trace 展示轮函数细节。

豊

哈希与派生

SHA-1/SHA-256/SHA-3/RIPEMD-160 摘要算法，HMAC 消息认证与 PBKDF2 密钥派生，全部基于自写哈希实现。

📄

编码转换

Base64 编解码与 UTF-8 字符编码，数据格式互转工具，所有编解码逻辑从零实现。

🔐

公钥与签名

RSA 密钥生成与 PKCS#1 签名、RSA-SHA1 签名方案、ECC-160 椭圆曲线运算、ECDSA 数字签名与验签。

📈

两级 Trace 追踪

Level 1 提供所有算法执行摘要（输入/输出/耗时），Level 2 展示 AES/SHA-256/RSA/ECDSA 等算法的详细中间步骤，使执行过程透明可审计。

🌐

RESTful API

全部算法通过 /api/* 端点暴露为 JSON 接口，统一响应格式（status_code / result / trace / time_ms），支持第三方程序调用。

✅

自动化自检

内置标准测试向量自检引擎，覆盖 FIPS 197、FIPS 180-4、RFC 4231 等权威测试数据，一键验证所有算法正确性。

🛠

从零实现

所有核心密码算法均从零手写实现，不依赖任何第三方密码库（如 PyCryptodome、cryptography），完全自主可控。
