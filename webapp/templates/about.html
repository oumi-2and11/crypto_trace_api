关于项目

本项目是网络空间安全编程技术与实例开发课程的期中项目，旨在构建一个密码算法接口化实现与可测验证平台。

平台基于 Python + Flask 开发，所有核心密码算法均从零实现（不依赖第三方密码库），并通过 RESTful JSON API 暴露接口，同时提供浏览器端可视化操作界面。

系统架构

平台采用四层架构设计：

  • 前端层：Jinja2 模板 + AJAX，三级模板继承（layout → layout_category → 页面）
  • API 层：Flask Blueprint + 统一响应封装（status_code / result / trace / time_ms）
  • 算法层：utils/ 下按类别组织，每个算法独立文件，从零实现
  • 公共层：统一编解码、参数校验、TraceCollector、耗时统计

Trace 追踪策略

支持两级 Trace 追踪，使算法执行过程透明可审计：

  • Level 1：所有算法通用摘要 — 算法名、操作、输入摘要、参数、输出摘要、耗时
  • Level 2：关键算法详细中间步骤 — AES 轮状态采样、SHA-256 消息调度与寄存器采样、RSA 密钥生成与签名步骤、ECDSA 点运算过程
